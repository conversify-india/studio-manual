Core Skill 2 · Document Spacing
Copy-Pasting & Paragraph Rules

Pasting text incorrectly will damage templates, break margins, and ruin font consistency.

❌ Unfiltered Paste (Ctrl+V)
- Pastes HTML background styles, font sizes, colors, and line spacing.
- Damages Google Doc template styles.
- Requires manual cleanup of every line.
- Double Enter carriage returns create excessive, irregular gaps.
✅ Plain Text Paste (Ctrl+Shift+V)
- Use Ctrl + Shift + V (Windows) or Cmd + Shift + V (Mac).
- Instantly adopts target template's fonts and style.
- To add vertical space: Go to Format > Line & paragraph spacing > Add space after paragraph.
- Neatly separate turns with exactly one blank line.
💡 Pro Tip: Never hit the Enter key twice to create paragraph spacing. Always use paragraph padding settings to keep the layout uniform.

Core Skill 5 · OCR Scan
ABBYY FineReader OCR Conversion

Used to convert scanned PDFs or static screenshots into digital, selectable layouts for translations.

🔍 What is ABBYY FineReader?
It is Optical Character Recognition software. It reads text layout structures inside scans, PDFs, or image captures, converting them into editable documents (like Microsoft Word or Excel).
📝 OCR Step-by-Step SOP
1. Launch ABBYY FineReader.
2. Click Open PDF/Image Document.
3. Set document language (Arabic, Persian, etc.) in options.
4. Click Recognize to extract layout.
5. Copy and paste as plain text into Google Doc.
⚠️ Verification Rule (Critical): ABBYY will highlight characters with low-confidence recognition in blue or red. Review these specific highlighted blocks side-by-side against original images and correct character reading errors manually.
Core Skill 6 · Audio Comments
Audio Listening & Google Doc Comments

Standard operations for handling complex, dialect-heavy speech segments and unverified words.

Wear Headphones
Always use over-ear headphones when transcribing or reviewing audio tasks. Avoid open speakers.
Slow Down Playback
If speaker dialect is fast, slow playback speed to 0.75x or 0.8x in your media player.
The Three-Times Rule
Play difficult audio segments at least three times at different volumes/speeds before tagging it.
Google Doc Comment Tags
Use Ctrl+Alt+M to insert tags. Format comments using these standard tags:
• [UNCLEAR @ MM:SS] - muffled speech or overlapping dialogue.
• [TERM QUERY] - jargon queries requiring client check.
• [PROPER NOUN] - unverified phonetic spelling of names/places.
Module 4 · Operations Glossary
Key Operations Terms Glossary

Terminology used in daily document and project file management.

File
PDF & DOCX
Portable Document Format (static) and Word Document (editable). We convert static PDFs to DOCX templates to translate while maintaining layout style.
Process
OCR (Optical Character Recognition)
Running software like ABBYY to read and extract textual layout blocks from scanned documents.
Process
DTP (Desktop Publishing)
Placing translated texts back into original design layouts (Illustrator/Photoshop) to align fonts and formats.
Tool
TM (Translation Memory)
A database tool storing previously approved translations to speed up recurring phrase checks.
Tool
Aura
LingoChaps' in-house platform at lingochaps.com for handling translation pipelines and AI workflows.
Quality
Track Changes
Word/Doc mode to show edits visually so translators see spelling and phrasing corrections clearly.
