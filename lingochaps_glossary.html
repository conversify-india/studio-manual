Translation terms — your quick reference

Every term you will hear at LingoChaps, explained simply. Click a category to filter.

All terms File formats Process Tools Client terms Quality
SRT
SubRip Subtitle
The most common subtitle file. Contains text with timestamps telling the video player when to show each line.
"The client sent an SRT — translate the text, never touch the timestamps."
File format
VTT
Web Video Text Tracks
Same as SRT but used on websites and HTML5 players. Treat it exactly like an SRT file.
"YouTube accepts SRT and VTT — check which format the client needs back."
File format
DOCX
Microsoft Word Document
The main editable format we deliver translated documents in. Clients can open and edit it in Microsoft Word.
"Convert to DOCX before sending — the client needs to edit it."
File format
PDF
Portable Document Format
Fixed-layout file that looks the same on every device. Clients often send source files as PDFs. We translate and usually deliver as DOCX unless asked otherwise.
"Source is a PDF — extract text, translate, rebuild layout in DOCX."
File format
EPS / AI / CDR
Vector design files
Design files from Adobe Illustrator (AI / EPS) or CorelDRAW (CDR). Contain text inside graphics — the DTP team handles these after translation.
"The brochure is an EPS — send to DTP once the text translation is done."
File format
DWG / DXF
CAD Drawing Files
Technical drawing files from AutoCAD used in engineering projects. Text labels inside the drawings need translating — the drawing structure must never be changed.
"Client sent DWG files — only translate the text labels, never move anything."
File format
OCR
Optical Character Recognition
Technology that reads text from scanned images or image-based PDFs. If a PDF is scanned (not native), we run OCR first to extract the text before translating.
"This PDF is scanned — run OCR first before we can extract the text."
Process
DTP
Desktop Publishing
Placing the translated text back into the original design — matching fonts, spacing, and layout so the final file looks like the original document.
"Translation done — send to DTP to put the text back into the layout."
Process
TM
Translation Memory
A database that saves previously approved translations. When the same sentence appears again in a future project, TM suggests the old approved translation automatically.
"Check the TM first — this client has lots of repeated sentences."
Process
OST
On-Screen Text
Any text that appears visually on a video — titles, lower thirds, graphic captions. Needs to be translated separately from dialogue and often requires DTP work to replace it.
"The video has OST in the corner — flag to client, we need the editable video file."
Process
Localisation
More than just translation
Adapting content for a specific region — language, dates, currencies, units, cultural references. Example: changing £ to $ and UK spellings to US spellings.
"Full localisation needed — translate AND adapt date formats and currency."
Process
Transcreation
Creative re-writing across languages
Used for marketing. Instead of translating, you recreate the message so it has the same emotional impact. A pun in English needs a new pun in the target language — not a literal translation.
"This ad slogan needs transcreation — direct translation would make no sense."
Process
Source / Target
The two languages in every job
Source = original language the document is written in. Target = language we are translating into. Written as Source→Target. EN→DE means English to German.
"Language pair is EN→FR — English is the source, French is the target."
Process
CAT tool
Computer-Assisted Translation
Software that helps translators work faster. Splits the document into segments, shows TM suggestions, and keeps terminology consistent. Examples: SDL Trados, memoQ, Phrase.
"Our translators use CAT tools — make sure they deliver in our required format."
Tool
Aura
LingoChaps translation platform
Our in-house tool at lingochaps.com for uploading, translating, and downloading documents. It uses an AI translation pipeline. Still being developed — report any issues to the tech team.
"Upload the file to Aura, select the target language, download the result."
Tool
Subtitle Edit
Free subtitle editing software
The tool we use for SRT and VTT files. Free to download. Shows timestamps, character count per line, and lets you play the video alongside the subtitles to check sync.
"Open the SRT in Subtitle Edit — character count shows in real time."
Tool
LSP
Language Service Provider
That is us — LingoChaps. Any company that provides translation, interpretation, or localisation services professionally is called an LSP.
"Client brief says 'send to your LSP' — that means send it to us internally."
Client term
Brief
Job instructions from the client
The document or message describing what the client needs — language pair, tone, deadline, file format, special instructions. Always read the full brief before starting.
"Brief says formal tone, Brazilian Portuguese — confirm both with the translator."
Client term
Word count
How we measure job size
Translation is priced by word count, not page count. Always run word count before quoting. In Word: Review tab → Word Count.
"Client wants a quote — Word Count first, then calculate rate × words."
Client term
Glossary
Approved terms list
A list of specific words that must always be translated the same way. Always share the glossary with the translator before they start — never after.
"Share the client glossary first — their product names stay in English."
Client term
QA
Quality Assurance
The final check before anything goes to the client. File name correct? Format right? No missing pages? Translation matches the brief? QA is done by the associate — it is not the same as review.
"QA check before sending — correct name, right format, no missing pages."
Quality
Back-translation
Translating back to check accuracy
Translating the translated text back into the source language to verify the meaning was preserved correctly. Used on medical and legal documents where accuracy is critical.
"Client wants a back-translation — translate the German version back to English."
Quality
Track changes
Editing with a visible trail
A Word feature showing every edit — who changed what and when. Reviewers must always use Track Changes so translators can see exactly what was corrected and why.
"Always turn on Track Changes before reviewing. Ctrl+Shift+E in Word."
Quality
Native speaker
Why it matters for quality
Someone who grew up speaking a language as their first language. We always hire native speakers of the target language — not people who learned it as a second language.
"For French translation — hire a native French speaker, not a learner."
Quality
